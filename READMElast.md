# CS2 ValuEyes · CS2 价值眼

> 不只是查价格——帮你判断"值不值"。

基于 FastAPI 的 CS2 饰品价值分析平台。集成 Buff / Steam / 悠悠有品等多平台实时价格，提供**自研估值模型 v2.0**、**AI 投资分析**、**挂刀计算器**、**开箱回报率排行**、**价格走势图** 五大核心功能。

## 快速启动

### 方式一：一键启动（推荐）

```bash
双击 start.bat
```

脚本会自动完成：
- 检查 Python 环境
- 创建虚拟环境 `.venv`
- 安装依赖
- **检测空闲端口**（从 8000 开始，被占用自动递增）
- 启动服务并打开浏览器

### 方式二：手动启动

```bash
pip install -r requirements.txt
python csqaq_api.py
```

访问 http://localhost:8000/ui

> 如果提示 API Token 无效，在 [CSQAQ](https://csqaq.com/) 注册获取 Token 后，替换 `csqaq_api.py` 中的 `API_TOKEN` 即可。

### 估值模型 v2.0（自研）

估值从四个维度综合计算，配有 **±30% 安全限幅器**，防止极端偏离市场价：

```
炼金基价 × 存世稀缺 × 市场深度 × 趋势健康 = 我的估值
```

| 维度 | 说明 | 系数范围 |
|:---|:---|---:|
| **炼金基价** | 品质 × 磨损系数，StatTrak ×1.4 / 纪念品 ×1.2 | ¥1 ~ ¥1000+ |
| **主战溢价** | AK-47/AWP/USP ×1.5，M4A1/M4A4 ×1.3 | ×1.0 ~ ×1.5 |
| **存世稀缺** | 基于实际存世量数据，越少越稀缺 | ×1.0 ~ ×5.0 |
| **市场深度** | Buff 求购/在售比 + Steam 深度 + 跨平台价差 | ×0.8 ~ ×1.5 |
| **趋势健康** | 超跌(< -30%)→溢价，暴涨(> 50%)→折价 | ×0.8 ~ ×1.2 |

```
估值判断：
  估值/市价比 > 120%  → 🟢 被低估
  估值/市价比 > 108%  → 🟡 轻微低估
  估值/市价比 < 80%   → 🔴 被高估
  估值/市价比 < 92%   → 🟠 轻微高估
  其余                → ⚪ 合理区间
```

### AI 投资分析

内置 **规则引擎 + DeepSeek LLM** 双模式：

**规则引擎（无需 API Key，自动运行）：**

6 条核心判断规则，从多维度综合评分（/100）：

| 规则 | 条件 | 评分影响 |
|:---|:---|---:|
| 短期过热 | 7天涨幅 > 30% | -20分 |
| 深度回调 | 7天跌幅 > 20% | +15分 |
| 补涨空间 | 大盘跑赢个品 > 10% | +15分 |
| 中期超跌 | 30天跌幅 > 25% | +10分 |
| 挂刀价值 | 挂刀比例 < 0.8 | +10分 |
| 流动性 | Buff 在售 > 1000 件 | +5分 |

```
综合评分 ≥ 70 → 🟢 推荐买入
综合评分 ≥ 50 → 🟡 观望
综合评分 ≥ 30 → 🟠 谨慎
综合评分 < 30 → 🔴 不推荐
```

**LLM 润色（可选，需配置 DeepSeek API Key）：**

配置后规则引擎的判断结果会由 DeepSeek 生成完整投资分析报告，从短期走势、中期趋势、大盘对比、流动性、性价比 5 个角度展开。

### 挂刀行情计算器

```
挂刀比例 = Buff售价 ÷ Steam实际到账    比例越低越划算
实际到账 = Steam求购价 - 手续费 ($0.01 ≈ ¥0.14)
```

- 支持 5% / 10% 费率切换
- 按挂刀比例升序排列
- 显示 Steam 到手余额、收益率、日成交量
- 支持价格区间和平台筛选

## 稳定性优化

### 一键启动脚本 `start.bat`

- 自动检测 Python 环境（支持 `py` Launcher 回退）
- 首次运行自动创建虚拟环境 + 安装依赖
- **自动检测空闲端口**（8000→8100 自动递增）
- 自动打开浏览器
- 停止后按任意键关闭窗口，不会闪退

### 端口自动检测

```bash
python csqaq_api.py --port 8080
# 或
python csqaq_api.py -p 8080
```

- 手动指定端口（不指定默认 8080）
- `start.bat` 双击运行时自动寻找空闲端口

### IP 绑定机制

- **先绑定 IP，再预热数据**，避免 API 401 错误
- 每次启动自动调用 CSQAQ 的 IP 白名单绑定接口
- 绑定失败时打印明确日志，不再静默吞错误
- 区分 `429` 频率限制和真正的绑定失败

### 大盘指数

- **启动时自动预热**大盘数据（饰品指数 K 线）
- 大盘获取失败时打印具体原因（超时/连接异常/API 错误）
- 失败后自动重试（缓存 5 分钟过期）
- 前端页面顶部实时显示大盘指数

### 错误处理

- **前端请求 15 秒超时**，超时后显示"请求超时，请稍后重试"
- 区分网络异常、HTTP 错误、API 业务错误，分别给出可读提示
- 后端捕获 `requests.Timeout` / `ConnectionError` 返回具体错误信息
- API Token IP 未绑定时提示"请重启服务"

### 图片加载

- 饰品图片使用 `loading="lazy"` 懒加载，不阻塞页面渲染
- 图片加载失败时显示"无图片"占位框，不影响价格等核心信息展示

## 配置 DeepSeek LLM（可选）

设置环境变量后重启即可：

```bash
set DEEPSEEK_API_KEY=你的deepseek密钥
python csqaq_api.py
```

不配置不影响核心功能，规则引擎和估值模型正常运行。启动时会有提示：

```
✅ DEEPSEEK_API_KEY 已配置，AI 分析将使用 LLM 润色
⚠️ DEEPSEEK_API_KEY 未设置，AI 分析仅使用规则引擎
```
