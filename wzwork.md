# 室友（王泽）工作成果 —— CS2 饰品 ML 价格预测模型

## 概述

基于 **RandomForest 机器学习算法**，对 CS2 饰品市场的未来 30 天涨跌幅进行预测。从约 **3.8 万条** 爬取数据中学习价格规律，综合价格、市场深度、存世量、历史波动、大盘走势等 **24 维特征** 进行建模。

---

## 核心成果

### 1. ML 价格预测模型（`price_predictor.py`）

| 项目 | 详情 |
|:---|:---|
| 算法 | RandomForestRegressor（400 棵树，最大深度 12） |
| 训练数据 | skins_full.csv + skins_raw(1).csv 合并，约 3.8 万条 |
| 特征维度 | 24 维（价格、深度、存世、稀有度、历史波动、大盘对比等） |
| 验证指标 | R² 评分 |
| 模型文件 | `price_model.pkl`（训练完成后序列化存储） |

### 2. 特征工程体系

从原始爬取数据中提取了 24 维特征，分为 6 大类：

| 类别 | 特征 | 说明 |
|:---|:---|---:|
| **价格特征** | `log_price` | 对数价格（降低长尾分布影响） |
| **市场深度** | `buy_sell_ratio`, `total_volume` | 求购/在售比、总成交量 |
| **存世稀缺** | `log_inventory`, `has_inventory` | 存世量对数、是否有存世数据 |
| **短期动量** | `rate_7` | 近 7 天涨跌幅 |
| **跨平台溢价** | `steam_premium` | Steam 与 Buff 的价差比率 |
| **同类对比** | `relative_to_peers` | 价格与同类（同品质+同磨损）中位数的比值 |
| **稀有度编码** | `rarity_idx`, `wear_idx` | 品质、磨损的有序编码 |
| **武器类型** | `w_rifle` ~ `w_other` | One-hot 编码（步枪/手枪/刀/贴纸/武器箱等 8 类） |
| **历史波动** | `hist_volatility`, `hist_price_slope`, `hist_short_term`, `hist_buy_volatility`, `hist_data_days` | 从 SQLite 历史价格数据提取的波动率和趋势 |
| **大盘对比** | `market_change_7d`, `market_change_30d` | 饰品市场整体指数变化 |

### 3. 预测评分系统

模型预测结果 + 规则引擎生成综合评分（0-100 分），并给出操作建议：

| 评分区间 | 判断 | 操作建议 |
|:---:|:---|:---|
| ≥ 70 | 📈 强烈看涨 | 建议买入 |
| 55 ~ 69 | 📈 看涨 | 建议关注 |
| 40 ~ 54 | ➡️ 中性 | 建议观望 |
| 25 ~ 39 | 📉 看跌 | 建议回避 |
| < 25 | 📉 强烈看跌 | 建议卖出 |

### 4. 多维信号分析

预测结果包含 7 类信号维度：

- 🧠 **模型预测** — ML 模型预测未来 30 天涨跌幅
- 💰 **市场深度** — 求购/在售比判断买盘强弱
- 📦 **存世稀缺** — 存世量稀缺程度评分
- 📊 **大盘对比** — 个品与大盘的涨跌偏离度
- 🔍 **同类对比** — 价格与同类中位数比较
- 📉 **历史波动** — 价格波动稳定性评估
- 🔄 **综合评分** — 加权汇总为 0-100 分

## 关键技术亮点

1. **合并训练策略** — 融合两份爬取数据，自动去重、补全缺失字段，最大化利用数据
2. **涨跌幅预测** — 直接预测未来 30 天涨跌幅（回归任务），而非简单的涨/跌分类
3. **加权训练** — 上涨样本权重 1.8 倍，使模型更关注有上涨潜力的饰品
4. **大盘对比特征** — 引入饰品市场整体指数，识别"跑赢/跑输大盘"的品种
5. **历史波动特征** — 从 SQLite 价格历史中计算波动率、价格斜率等时序特征
6. **智能模糊搜索** — 后端智能分词支持中英文混合搜索（如"液化mp5"），前端防抖 + 高亮 + 分页 + 通用组件化设计
7. **优雅降级** — 模型不存在时自动训练；大盘数据失败时使用缓存；全链路异常保护
